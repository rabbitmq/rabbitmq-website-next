Java Libraries

This page lists support and compatibility information for the Java libraries maintained by the RabbitMQ team.

RabbitMQ Java Libraries Support Timeline
Library and Branch General Support Until 1 Extended Support 2 JDK Version Range 3
Java Client 5.x Currently supported 8-17
Java Client 4.x 31 July 2020 31 December 2020 6-8
JMS Client 2.x Currently supported 8-17
JMS Client 1.x 31 July 2020 31 December 2020 6-8
PerfTest 2.x Currently supported 8-17
Hop 3.x Currently supported 8-17
Reactor RabbitMQ 1.x Currently supported 8-17

(1) Includes backport of features (if possible), bug and security fixes for non-latest production branches.

(2) Includes only security patches. The library reaches end of life (EOL) once the extended support period ends.

(3) Long-Term Support (LTS) releases of the JDK are fully supported (currently JDK 8 and 11). Intermediate releases (e.g. 9 and 10) are supported on a best-effort basis, meaning bug reports are accepted and addressed as far as technically possible but without any service level guarantees.
